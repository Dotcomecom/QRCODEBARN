QRCodeBarn Production SEO Pack
================================
Upload all files in this ZIP to the public_html folder for qrcodebarn.com.

Included:
- Homepage free QR code generator
- 16 SEO landing pages
- About, FAQ, Privacy, Terms, Contact
- Shared CSS and JavaScript
- Working QR generator on every tool page
- PNG and SVG download buttons
- FAQ, Breadcrumb, WebApplication and HowTo JSON-LD schema
- Canonical tags
- Open Graph and Twitter metadata
- Internal linking/topic clusters
- AdSense-ready placeholder blocks
- sitemap.xml and robots.txt

After upload:
1. Visit https://qrcodebarn.com/sitemap.xml
2. Submit sitemap in Google Search Console
3. Replace AdSense placeholder blocks with real ad units when approved
4. Test QR generation on desktop and mobile
