Frequently asked questions

What is a QR code for this use case?

It is a simple shortcut that sends a person to a relevant page, document or action without forcing them to type anything.

Do I need to test it before publishing?

Yes. Always test the code on more than one phone and from the expected print distance before using it in a live setting.

Can I use it on print materials?

Yes. QR codes are commonly placed on menus, flyers, cards, signs, packaging and labels when the destination is clear.
