QR Code Error Correction

QR codes include error correction, which helps them scan even when part of the code is slightly damaged, dirty or covered. Higher error correction can make a code more resilient, but it can also increase visual complexity.

Error correction is especially important when adding a logo to the centre of a QR code. Keep the logo modest, use strong contrast, and test the code on several phones.

Do not rely on error correction to rescue a poor design. A clean, high-contrast QR code without unnecessary decoration will usually scan better than an over-designed version.
