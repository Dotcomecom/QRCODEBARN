How to Create a QR Code

Creating a QR code is straightforward: choose the type of information, enter the details, generate the code, download it and test it before publishing.

Step 1: choose the right QR code type

Start with the job the code needs to perform. A website URL code is best for sending people to a page. A WiFi QR code is useful for guest access. A vCard code is useful for contact details. An email or SMS code can pre-fill a message.

Step 2: enter accurate information

Small mistakes can make a QR code useless. Check spelling, phone numbers, email addresses, URLs, WiFi network names and passwords carefully before generating the code.

Step 3: generate and download the QR code

Use a clear output format suitable for your purpose. PNG files are convenient for websites and documents. SVG files are useful when a code needs to be scaled for print without losing sharpness.

Step 4: test before publishing

Test the code on at least two phones, ideally one iPhone and one Android device. Test it at the size and distance people will actually use.

For printed materials, print a sample before ordering a large batch. Testing is much cheaper than reprinting business cards, posters, menus or packaging.

Step 5: place it with context

Do not place a QR code on its own without explaining what it does. A short instruction such as “Scan to view the menu” or “Scan to save our contact details” gives users confidence.
