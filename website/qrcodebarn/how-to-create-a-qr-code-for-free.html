How to Create a QR Code for Free

Creating a QR code for free is straightforward if you know what destination you want.

Quick answer

  • Choose the QR code type first.
  • Enter the correct destination or data.
  • Download and test before printing.

Step 1: choose your QR type

Decide whether the QR code should open a website, display text, start an email, call a phone number, send an SMS or share WiFi details.

Step 2: enter the data

Type the destination carefully. For website QR codes, include the full URL.

Step 3: customise the code

Choose sensible colours and size. High contrast is more important than decorative design.

Step 4: test the code

Scan the QR code on at least two devices before you print or publish it.
